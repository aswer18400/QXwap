Naming & Usage Rules

Figma namesใช้รูปแบบ Icon / Search, Icon / Xwap, Icon / Shipment.
Code namesใช้ lucide-react export เช่น Search, Bell, HeartHandshake.
SVG exportใช้ lowercase kebab เช่น icon-search.svg, icon-xwap.svg.
AccessibilityIcon-only button ต้องมี aria-label. Decorative icon ใช้ aria-hidden.
