QXwap Journey High Wireframes

High-wireframe สำหรับ flow หลักของ QXwap จาก UI/logic ปัจจุบัน ใช้เป็น journey map ใน Figma ก่อนลงรายละเอียด visual. ทุก flow โฟกัสว่า user เข้าใจอะไร, action ถัดไปคืออะไร, และจุดไหนต้องเชื่อมกับ API/state เดิม

Frame: mobile 390x844 Primary CTA: Xwap Guest → Login → Offer Search + Filter Add Product + Upload Inbox + Shipment
