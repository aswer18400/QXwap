2. Component Specs จากโค้ดปัจจุบัน

รายการ component ที่ต้องทำเป็น Figma component และ state ที่ต้องมี

TopbarCode: apps/web/src/components/Topbar.tsx
Classes: .topbar, .brand, .icon-btn, .avatar-btn
Icons: Search, Bell, User
BottomNavCode: BottomNav.tsx + NavButton.tsx
Icons: Home, ShoppingBag, Plus, Inbox, User
States: default, active, safe-area
AiFeedCode: screens/AiFeed.tsx
Classes: .ai-feed, .follow-rail, .credit-card, .feed-market-card
Icons: Sparkles, Bookmark, Repeat2
ProductGridCardCode: components/ProductGridCard.tsx
Classes: .shop-card, .shop-photo, .save, .condition-pill
Icons: Bookmark, BookmarkCheck, HeartHandshake
DetailCode: screens/Detail.tsx
Classes: .detail, .hero-img, .owner-panel, .info-grid
Icons: Bookmark, Edit3, Trash2, HeartHandshake
SheetsCode: sheets/*.tsx
Classes: .sheet-backdrop, .sheet, .sheet-head
Icons: X, Send, Truck, Camera, Search
InboxCode: screens/InboxPage.tsx
Classes: .segmented, .offer-panel, .offer-top
States: offers, notifications, pending, accepted, rejected
ProfileCode: screens/Profile.tsx + ProfileShop.tsx
Classes: .profile-card, .profile-menu, .level-card, .profile-product-tabs
WalletCode: screens/WalletPage.tsx
Icon: WalletCards
States: logged out, loaded balance, transaction list
