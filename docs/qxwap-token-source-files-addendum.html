Design System Files

ไฟล์เหล่านี้คือ source/reference ที่ Figma และ developer ต้องใช้ร่วมกัน.

Token sourcetokens.jsonapps/web/src/design-system/tokens.json

Primitive, semantic, component tokens. ใช้ import เข้า Figma/Tokens Studio ได้.

CSS variablesqxwap-design-tokens.cssapps/web/src/design-system/qxwap-design-tokens.css

ตัวแปร `--qx-*` สำหรับ colors, spacing, radius, typography, shadow, motion, layout.

Componentsqxwap-components.cssapps/web/src/design-system/qxwap-components.css

Reference component classes แบบ prefix `qx-` สำหรับ state/spec examples.

Tailwindqxwap-tailwind-preset.jsapps/web/src/design-system/qxwap-tailwind-preset.js

Preset ที่ map Tailwind theme กลับไป QXwap tokens.

Examplescomponents-example.htmlapps/web/src/design-system/components-example.html

ตัวอย่าง card, nav, form, login, feed item, shop grid.

DocsREADME.mdapps/web/src/design-system/README.md

Production rules, accessibility, dark mode, usage examples.
